# Creative Brief: 7th Sub-Plane
## Visual Audio Text Media Applications

---

## THE CONCEPT
The 7th Sub-Plane is **The Basement** of reality - not metaphorically, but literally the densest, lowest vibration of astral matter located within the Earth's crust. It is the "Slag" to the 6th Plane's "Mirage."

**Core Aesthetic**: Dystopian Ether - Industrial Occult - Viscous Horror

---

## VISUAL DIRECTION

### Color Science
**Primary Palette**:
- Dried Blood (#3E1E1E) → Void-Black (#1A0A0A)
- Lurid Red (#7F0000) - the color of cruelty and clotted blood
- Brownish-Grey - wet ash, the tint of selfishness

**Lighting Rules**:
- NO shadows (self-luminous matter)
- NO traditional light sources
- Illumination from: Black Fire (un-lights), Livid Violet phosphorescent veins
- Visual Burn-In: Objects leave black silhouette ghosts

**Material Library**:
| Material | Description |
|----------|-------------|
| Semi-Solidified Tar | Ground texture, viscous, semi-solid |
| Vitrified Obsidian | Under-earth cities, impossibly smooth |
| Metallic Slime | Entity surfaces, sticky, reflective |
| Soot-Mist | Atmospheric particulate, flecks vision |
| Rusted Iron | Structural elements, #3E1E1E base |

### Entity Design Language
- **Human Dregs**: Bloated, animalized, predatory gleam, no ego-light in eyes
- **Shells**: Uncanny valley plastic/wax, hollow eyes, staccato movement
- **Gnomes**: Geometric, crystalline eyes, anthracite skin, no faces
- **Artificials**: Jagged geometry, barbed hooks, amoebic slime
- **Shadows**: Perfect 3D silhouettes of pure black, light-absorbing

### Environmental Motifs
- **Claustrophobia**: No horizon, low ceiling, inside solid rock
- **Industrial Organic**: Fungoid lichen, sweating walls, tar-glass
- **Tectonic Scale**: Cyclopean architecture, resonance chambers, deep time

## AUDIO DIRECTION

### Ambient Bed
**The Constant Grind**:
- Low-frequency (20-60Hz) metal-on-metal tectonic grinding
- Layer: Bubbling liquid (simmering pot thickness)
- Layer: Sub-audible hum (electrical transformer)

### Speech Design
**Syllable-Lag Effect**:
- Viscous, muffled, elongated
- As if speaking through vacuum or thick liquid
- Delay between words: 0.5-1 second
- Reverb: Massive, underground, claustrophobic

### Entity Sounds
| Entity | Sound Signature |
|--------|-----------------|
| Hate-Hooks | High metallic hiss, steam escape |
| Vampiric Blobs | Wet suction, organic slither |
| Gnomes | Tectonic grinding, rock pressure |
| Regents | Sub-bass hum, mountain-crushing weight |
| Deva-Dregs | Cracked crystal bell, mournful |

### Dynamic Events
- **Desire-Storms**: Black cyclone audio - grit scratching, red spark crackles
- **Second Death**: Shattering glass + relief sigh + vacuum silence
- **Phase-Shift**: Gas-to-solid transition sound (wisp to claw)

## TECHNICAL SPECIFICATIONS

### Shader Parameters
```glsl
// 7th Plane Material
float viscosity = 0.9; // Movement resistance
float selfLuminance = 1.0; // No external lights
float opacityPlus = 2.0; // Aggressive light absorption
vec3 sootColor = vec3(0.118, 0.039, 0.039); // #1E0A0A
float particulateDensity = 0.7; // Soot-mist flecks
```

### Physics Simulation
- **Gravity**: 2x normal, magnetic pull downward
- **Friction**: High resistance on all movement
- **Fluid Dynamics**: Thick oil/mercury viscosity
- **Particle Systems**: Microscopic grit sandblasting

### Post-Processing Stack
1. **Chromatic Aberration**: Heavy on edges (vibration distortion)
2. **Film Grain**: Soot particulates
3. **Vignette**: Claustrophobic closing
4. **Color Grading**: Lift shadows to #3E1E1E, crush blacks to #1A0A0A
5. **Bloom**: Only on livid violet and black fire sources
